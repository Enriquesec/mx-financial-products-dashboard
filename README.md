# MX Banking Insights

Análisis y visualización de información sobre tarjetas de crédito y cuentas de captación en el sistema bancario mexicano.

Este proyecto tiene como objetivo centralizar, estructurar y presentar datos relevantes de productos financieros ofrecidos por bancos y fintechs en México, con énfasis en comparativos claros y visualizaciones accesibles.

---

## Objetivos

- Presentar información comparativa de tarjetas de crédito en México.
- Analizar la participación porcentual de cuentas de captación por institución.
- Facilitar la exploración de datos mediante una interfaz web.
- Servir como base para análisis financiero, educación financiera y proyectos de data analytics.

---

## Instituciones consideradas

El proyecto contempla bancos tradicionales y fintechs, entre ellos:

- BBVA
- Santander
- Banorte
- Citibanamex
- HSBC
- Nu
- Mercado Pago
- Hey Banco
- Otras instituciones relevantes del sistema financiero mexicano

La lista puede ampliarse conforme se integren nuevas fuentes de información.

---

## Tipos de información

### Tarjetas de crédito
- Tipo de tarjeta
- Anualidad
- CAT
- Beneficios principales
- Segmento objetivo

### Cuentas de captación
- Tipo de cuenta
- Participación porcentual por institución
- Comparativos entre bancos y fintechs
- Tendencias agregadas

---

## Fuentes de datos

Las fuentes incluyen información pública y agregada, como:

- Reportes de la CNBV
- Información del Banco de México
- Datos publicados por las instituciones financieras
- Documentación y comunicados oficiales

Este proyecto no utiliza información personal ni datos confidenciales.

---

## Stack tecnológico

- Frontend: HTML, CSS, JavaScript
- Visualización: Chart.js o D3.js
- Data y análisis: Python, Pandas, Jupyter Notebooks
- Hosting: GitHub Pages
- Control de versiones: Git y GitHub
